# ShieldHer

## Environment Setup
0. Use Node.js 20.19.4+ (LTS recommended). Expo SDK 54 and Metro in this project rely on modern APIs that are not available in Node 18.

1. Copy the example environment file:

```bash
cp .env.example .env
```

2. Replace placeholder values in `.env` with your project configuration.
3. Never commit `.env` to version control.
